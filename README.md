# CATSUBA IS STILL UNDER DEVELOPMENT.
Catsuba is not suitable for production. There are numerous bugs, unfinished features, and flaws that are yet to be addressed. Feel free to contribute.

## Catsuba

Catsuba is an imageboard software written primarily in PHP and some JS. It is made to be configurable and out-of-the-box,
with support for SQLite and MySQL/MariaDB, an escape-friendly HTML rendering engine, among other features.
